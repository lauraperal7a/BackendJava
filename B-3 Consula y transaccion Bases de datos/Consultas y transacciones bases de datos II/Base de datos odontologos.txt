Base de datos odontologos 

Crear una entidad Odontologos en la base de datos H2 que tenga 
los siguientes campos:
apellido: String
nombre: String
matricula: String

Para ello, te pedimos crear una connection a la base de datos e 
insertar una fila sobre la tabla odontólogos y luego hacer un update 
que le cambie la matrícula. Finalmente, no te olvidés corroborar con 
una consulta la inserción y la modificación.
